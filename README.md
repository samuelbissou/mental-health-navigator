Overview 

The Mental Health Resource Navigator is a desktop application that helps users find mental health resources quickly and save their favorites for easy access.

Features 

1. Searching for Resources 

Type keywords in the search box (e.g., "crisis", "therapy") 

Click the "Search" button or press Enter 

Results will appear in the table below 

2. Filtering by Category 

Use the "Category" dropdown menu 

Select: All, Crisis, Information, Therapy, Online Resource, or Support Group 

Table automatically updates 

3. Emergency Crisis Support 

Click the red "CRISIS: Call 988" button at the top 

A popup shows emergency hotline numbers 

Available 24/7 

4. Viewing Resource Details 

Click any row in the table 

A popup shows full details: name, phone, website, description, cost 

5. Saving Favorites 

Select a resource by clicking a row 

Click "Add to Favorites" button 

Resource is saved to your favorites list 

6. Viewing Favorites 

Click "View Favorites" button 

Table shows only your saved favorites 

Click "Clear" to see all resources again

Click "Clear" button 

Returns to showing all resources
